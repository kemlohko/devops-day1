# Git Flow Lab

A simple web page used for practicing Git workflows.

## Features

- Header with navigation links
- Theme toggle button
- Light and dark themes

## Dark Theme

Click the **Toggle Theme** button in the header to switch between light and dark modes.

The implementation works by toggling a `data-theme="dark"` attribute on the `<body>` element. CSS attribute selectors apply the appropriate dark-theme styles when the attribute is present.
